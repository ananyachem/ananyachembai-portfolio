Malaria Cell Classification Using CNNs

2 min read.


Malaria is a life-threatening disease caused by Plasmodium parasites, and timely diagnosis is critical to treatment. In this project, I built a Convolutional Neural Network (CNN) that looks at a microscope image of a blood cell and decides: parasitized or uninfected. A binary classification problem, tackled with a custom residual CNN architecture — with the goal of making detection faster and more accurate than the human eye on a long shift.


Fig.1 Malaria Infected Cell

1. Data Augmentation

To improve the model's ability to generalize and handle variability in image orientation, lighting, and scale, I applied a rich set of transformations:
- Random horizontal and vertical flips
- Random brightness, contrast, saturation, and hue shifts
- Random rotations (0–270°)
- Random zooming followed by resizing and cropping
All of this helped mimic the real-world distortions you'd see in microscopic imaging.


2. CNN Architecture with Residual Blocks

I implemented a CNN with residual connections to tackle the vanishing gradients problem and boost performance on deeper networks. 


    Architecture Summary:

        Initial Conv Layer:
        32 filters, 3×3 kernel, ReLU activation
            Batch Normalization
            Max Pooling

        Residual Blocks:
            3 blocks with increasing filters: 64, 128, 256
            Each block contains:
                2 convolutional layers
                Batch normalization
                Shortcut/skip connection

        Fully Connected Layers:
            Flatten
            Dense(256) → ReLU → Dropout(0.5)
            Dense(128) → ReLU → Dropout(0.5)
            Output Layer: 1 neuron with sigmoid activation

        Training Setup:
            Optimizer: AdamW (lr = 0.0001, weight decay = 1e-4)
            Loss Function: Binary Cross-Entropy

Conclusions

Data augmentation gave the model serious robustness and kept overfitting in check, while the residual connections made training deeper layers painless. Even with a relatively small network, the architecture delivered state-of-the-art results for binary medical image classification.
With more data and fine-tuning, I truly believe this model could be part of a real-time diagnostic pipeline for malaria detection in low-resource settings — which is exactly where it's needed most.
